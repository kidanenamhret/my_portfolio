# Mesfin Portfolio (React + Vite)

![Demo Screenshot](file:///c:/Users/tadwo/Desktop/html/build_portfolio/react_portfolio/screenshot.png)

## ✨ Overview
A modern, glass‑morphism personal portfolio showcasing projects, live stats, and interactive UI built with React 18, Vite 8, Tailwind, and Boxicons.

![Boxicons](https://img.shields.io/badge/boxicons-2.1.4-000000?logo=boxicons)

## 🚀 Quick Start
```bash
npm install
npm run dev
```
Open <http://localhost:5173> in your browser.

## 🌟 Features
- **Dynamic Stats** (likes, messages, shares) with simulated real‑time updates.
- **Project Gallery** with search, tech‑filter tags, and live demo / code links.
- **Social Icons** tracking clicks via a mock Node.js logger.
- **Glass‑morphism UI** with dark background, gradients, and subtle micro‑animations.
- **Responsive Layout** – works on mobile, tablet, and desktop.
- **Future‑ready** – dark‑mode toggle, PWA support, and contact form can be added.
